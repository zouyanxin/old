## 1. elderly-background.jpg（背景图）

**用途**：网页整体背景图

**要求**：
- 尺寸：1920×1080px 或更大（建议2560×1440px）
- 格式：JPG（压缩后500KB以内）
- 内容：温馨的老年人生活场景
- 色调：柔和、温暖（避免过于鲜艳或暗沉）

**推荐内容**：
- ✅ 公园里散步的老年人
- ✅ 社区活动中的老年人
- ✅ 家庭聚会场景
- ✅ 老年人锻炼（太极拳、广场舞）
- ✅ 温馨的养老院场景
- ❌ 避免：医院、病床、悲伤场景

**图片来源建议**：
1. **免费图库**（需注意版权）：
   - Unsplash: https://unsplash.com/s/photos/elderly-chinese
   - Pexels: https://www.pexels.com/search/elderly/
   - Pixabay: https://pixabay.com/images/search/elderly/

2. **自己拍摄**（推荐）：
   - 拍摄家人或社区老年人（需获得授权）
   - 注意保护隐私，避免清晰面部特写

3. **购买版权图片**：
   - 视觉中国: https://www.vcg.com/
   - 站酷海洛: https://www.hellorf.com/

## 📐 图片优化建议

### 1. 尺寸优化
```
原始尺寸 → 目标尺寸 → 压缩
4000×3000 → 1920×1440 → 500KB以内
```

### 2. 质量设置
- JPG质量：80-85%（平衡质量和文件大小）
- 避免过度压缩导致模糊

### 3. 色彩调整
- 亮度：适中（避免过暗或过亮）
- 对比度：柔和（避免刺眼）
- 饱和度：自然（避免过于鲜艳）

---

## 🎨 色调参考

为了与网页整体风格协调，建议图片色调：

**主色调**：
- 蓝色系（#2c5aa0）：信任、专业
- 绿色系（#2da44e）：健康、活力
- 暖色系（#f5f8fa）：温馨、舒适

**避免**：
- 过于鲜艳的红色、紫色
- 过于暗沉的黑色、灰色
- 冷色调的纯白背景

## 📝 版权注意事项

### ⚠️ 重要提醒

1. **使用免费图库**：
   - 确认图片为CC0（公共领域）或允许商业使用
   - 部分图库要求署名，请遵守许可协议

2. **自己拍摄**：
   - 如拍摄他人，需获得肖像权授权
   - 避免侵犯隐私（如清晰的面部特写）

3. **购买版权图片**：
   - 保留购买凭证
   - 遵守使用范围限制

4. **禁止**：
   - ❌ 直接从百度图片等搜索引擎下载使用
   - ❌ 使用明显有水印的图片
   - ❌ 使用明星、名人照片（除非获得授权）

## 🚀 快速开始

### 如果暂时没有图片

1. **方案1**：使用纯色背景
   - 编辑 `styles.css`，注释掉背景图代码：
   ```css
   body {
       /* background-image: url('images/elderly-background.jpg'); */
       background-color: #f5f8fa;
   }
   ```

2. **方案2**：使用占位符服务
   - 临时使用在线占位图：
   ```html
   <img src="https://via.placeholder.com/800x600/2c5aa0/ffffff?text=老年人健康" alt="占位图">
   ```

3. **方案3**：先使用低质量图片
   - 快速找一张符合主题的图片
   - 后续再替换为高质量图片
